# 我的音乐品味

- 林俊杰
- 陈奕迅
- Taylor Swift
- Coldplay
- 邓紫棋
- 孙燕姿
- 五月天
- 苏打绿

## 喜欢的风格
- 华语流行
- 民谣
- 独立摇滚
- R&B
- 电子

## 年代偏好
- 主要听 2000-2020 年代的华语音乐
- 也听 2010 年代后的欧美独立音乐

## 黑名单（绝对不能推荐的艺人）
- 周杰伦 — 网易云无播放版权

## 其他偏好
- 喜欢歌词有深度的歌
- 工作时候倾向纯音乐或轻音乐
- 晚上喜欢安静的氛围音乐

## 天气偏好

- 雨天 -> 偏爱爵士、民谣、慢节奏
- 晴天 -> 偏爱欢快流行、独立摇滚
- 阴天 -> 偏爱电子、氛围音乐、R&B
- 雪天 -> 偏爱温暖的歌、治愈系
- 雾天/雾霾 -> 偏爱轻音乐、纯音乐
- 天气炎热（>30度）-> 偏爱清爽、清凉感的音乐
- 天气寒冷（<10度）-> 偏爱温暖、有厚度的歌
